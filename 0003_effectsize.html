  1. Magnitude of difference between groups.
  2. “You should describe the results in terms of measures of magnitude –not just, does a treatment affect people, but how much does it affect them.”

    -Gene V. Glass, on the importance of mentioning effect size.
  3. If statistical significance is denoted by p value, then practical significance (real world) is denoted by effect size.
    • P value = shows the existence of effect.
    • Effect size = shows the magnitude of effect.
  4. Effect size is not affected by sample size. It estimates a population parameter. Whereby, p value is influenced by sample size as well as effect size.
    • Effect size in the two studies is the same (r=0.30), although different N. Effect size is not influenced by sample size.
    • But only in study with N=100 is statistically significant (CI does not cross 0).
    • A population size of N=30 requires a value of r=0.35 to get p<0.05.
    • This shows: p is influenced by sample & effect size.
  5. Effect size and sample size:
    • Large effect size = possible to be detected in small sample size.
    • Small effect size = requires large sample size.
  6. Example of study that shows importance of effect size:
    • Aspirin reduces risk of MI, P<.00001
    • Therefore, it was statistically significant, and therefore recommended for prevention.
    • However, r2 = 0.001 → extremely small effect size! {r2 = coefficient of determination, a type of effect size}
    • As a result, many people took aspirin, although almost no benefit, while at risk of adverse effects.
  7. Another example:
    • The result is statistically significant. That means there is a difference in effect between the methods.
    • However, with an absolute difference of only 0.1 kg, is it really an effect? Can we just neglect it?
    • Moreover, p-value does not even tell which one gives the slightly better mean weight loss.
    • This is where calculating effect size (eg. using Cohen’s d) plays a role.
  8. Effect size is helpful to be calculated before and after data collection.
    • Before study:
      • Expected effect size can determine minimum sample size needed.
      • Sample size affects power of study (likelihood of rejecting false null hypothesis/likelihood of detecting an effect when there is one).
      • Do power analysis to determine minimum sample size.
    • After study done:
      • Report the actual effect size in our result.
  9. We can calculate effect size via Cohen’s d, Pearson’s r (correlation of coefficient), odds ratio, relative risk, and others.
